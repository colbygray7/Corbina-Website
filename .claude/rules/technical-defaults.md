# Technical Defaults

## Stack and Tools

- **Tailwind CSS**: Use via CDN (`<script src="https://cdn.tailwindcss.com"></script>`)
- **Placeholder Images**: Use `https://placehold.co/` when source images aren't provided
- **Design Approach**: Mobile-first responsive design
- **File Structure**: Single `index.html` file unless the user requests otherwise

## Code Quality

- Keep code clean but don't over-abstract
- Inline Tailwind classes are fine and preferred
- All content should be inline — no external files unless requested
